# 🚀 PortScanner-Pro

> **Advanced Multi-Threaded Network Port Scanner built with Python**

PortScanner-Pro is a professional Python-based network scanning tool designed to identify open TCP ports, detect running services, perform host discovery, grab service banners, and generate detailed scan reports. The project demonstrates practical networking concepts, socket programming, multithreading, and cybersecurity fundamentals.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/PortScanner-Pro.git
```

Go into the project

```bash
cd PortScanner-Pro
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
